API Endpoints

Acesso programático a milhares de rádios online

Endpoints Disponíveis

GET /api/popular Populares

Rádios mais populares (externas + customizadas)

GET /api/search Busca

Busca unificada com filtros avançados

GET /api/countries Países

Lista de países com contadores

GET /api/genres Gêneros

Gêneros musicais disponíveis

GET /api/external-radios/* Proxy

Proxy para Radio-Browser API
